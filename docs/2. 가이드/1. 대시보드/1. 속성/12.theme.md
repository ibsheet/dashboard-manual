# theme

## 타입

`string`

## 설명

대시보드의 테마를 설정합니다. 기본적으로 light 테마로 설정이 되어있으며, light 외에 dark 테마가 존재합니다.

## 사용 예제

```javascript
IBDashboard.create(document.getElementById("dashboard"), {
  options: {
    theme: "light", // 또는 "dark"
  },
  dataSets: dataSets1,
});
```

## 관련 API

- `changeTheme(theme)`: 테마 변경 API
