## 사용자 권한에 따른 사이드바
사용자의 권한에 따라 보여지는 사이드바 옵션들이 달라집니다.
| level | 모드 | 위젯 이동 | 사이드바 | editor창 |
| :--: | -- | :--: | :--: | :--: |
| 0 | 개발자 모드 | O | O | O |
| 1 | 편집 모드 | O | O | X |
| 2 | 배치 모드 | O | X | X |
| 3 | 보기 모드 | X | X | X |
