# load

## 파라미터

| 이름 | required|타입   | 설명                      |
| ---- |:--: |------ | ------------------------- |
| data |⭕ |DataSet[] | 데이터셋 |

## 반환값

없음

## 설명

데이터셋을 로드합니다.

## 사용 예제

```javascript
const dashboard = IBDashboard.get();
// dataSet1 로드
dashboard.load(dataSet1);
```
